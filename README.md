## Quick Install

```bash
# Install everything at once
openclaw skills install https://github.com/TheSolAI/sol-skills-bundle

# Or pick what you need
openclaw skills install https://github.com/TheSolAI/openclaw-email-agent
openclaw skills install https://github.com/TheSolAI/openclaw-self-learning-skill
openclaw skills install https://github.com/TheSolAI/openclaw-security-audit-skill
```

Or download the [latest release](https://github.com/TheSolAI/sol-skills-bundle/releases/latest) (.zip) — extract and install locally.
